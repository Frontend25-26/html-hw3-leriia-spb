Вопрос 1: HTML создал Тим Бернерс-Ли в ЦЕРНе в начале 1990-х.

Вопрос 2: Первое описание HTML было опубликовано в 1991 году на статусной странице CERN.

Вопрос 3: <p>, <a>, <img> — базовые теги с HTML 2.0. <section> появился только в HTML5.

Вопрос 4: WorldWideWeb.

Вопрос 5: Цвет #adefcb — это rgb(173, 239, 203), пастельный оттенок мятно-зелёного цвета. Если ты нашёл его в палитре — тоже хорошо!

Вопрос 6: HTML5 стал официальным стандартом 28 октября 2014 года.
